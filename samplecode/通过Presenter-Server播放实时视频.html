通过Presenter Server播放实时视频

通过Presenter Server播放实时视频

前提条件

已参考Enable the Web Service on the UI Host启动presenter server。

Presenter Server播放

在Mind Studio服务器中以hisilicon用户ssh登录到Atlas DK开发板中。

ssh hisilicon@192.168.0.2

说明：
192.168.0.2请替换为开发板实际IP地址。

在Atlas DK开发板上执行如下命令：

sudo ./ascendcamera -v -c 1 --fps 20 -t 0 -w 704 -h 576 -s 10.10.10.1:7006 /channelname

其中：

  • -s后面的值10.10.10.1为Presenter Server的IP地址（即3中“config.conf“中的“presenter_server_ip“的值）。
  • 7006为presenter server的端口号（即3中“config.conf“中的“presenter_server_port“的值）。
  • channelname为您在presenter server端配置的channel name，即为图2中配置的“Channel Name“，例如video。
  • 其他详细参数请执行./ascendcamera命令或者./ascendcamera --help命令参见帮助信息。
