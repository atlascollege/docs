Output Description

Output Description

  1. 登陆 Presenter Server 网站（仅支持Chrome浏览器），URL为http://web_server_ip:web_server_port**，配置查看请参见Enable the Web Service on the UI Host。

     

  2. 进行人脸注册。

    1. 点击“Face Library“按钮，输入“Username“。

    2. 单击“Browse“按钮，上传人脸图像（人脸图像裁剪时尽量按照“Example Photo“的比例设置）。

    3. 点击Submit按钮上传，若上传失败，可以更改裁剪比例。

  3. 人脸识别以及比对。

    在Present Server界面中点击对应的App Name，若有人脸出现在摄像头中，且与已注册人脸匹配一致，则会出现对应人员姓名及相似度的标注。
