Output Description

Output Description

  1. 登陆 Presenter Server 网站（仅支持Chrome），URL为http://web_server_ip:web_server_port**，配置查看请参见Enable the Web Service on the UI Host。

  2. 等待Presenter Agent传输数据给服务端，单击“Refresh“刷新，等待相应的channel 的Status变成绿色。

    说明：

    • 此处的Channel Name为3重配置的ChannelName。
    • Presenter Server最多支持10路channel同时显示，10路20帧是极限情况，受限于网络带宽的影响，帧率会自动适配为较低的帧率进行展示。

  3. 单击后面的Channel Name链接，比如上图的“video”，查看结果。

    • 对于图片模式，因为agent一般会立即退出，所以Status一般总是处于灰色状态，点开channel显示的上一次获取的人脸识别图片，如果channel从来没有图片输入，则显示空白。

    • 对于视频模式，点开相应channel后，如果有agent发送数据过来，则显示视频，如果没有数据，则显示空白。
