OSD

OSD

Description

OSD提供图像上的公共叠框能力，当前支持的输入图像格式是YUV420SP NV12格式，输出图像格式是YUV420SP NV12格式，提供四个能力：画矩形框、画矩形框和文字、画圆形框、写文字。

Sample中的使用方式是：人脸检测Demo经过CameraDatasets、ImagePreProcess、Inference Engine、PostProcessEngine，生成检测之后的人脸标记数据，PostProcessEngine调用osd的接口，将人脸标记数据以及视频中的原图像一帧帧传给osd，osd将人脸标记数据生成box，叠加到原始图像中，然后将叠加后的图像按照原格式返回给PostProcessEngine。

Sample Code

// construct rectangle position
RectanglePosition rectanglePosition;
// set left top point of rectangle
Coordinate anchorLt;
anchorLt.x = ptr[ANCHOR_LEFT_TOP_X_AXIS_INDEX] * width;
anchorLt.y = ptr[ANCHOR_LEFT_TOP_Y_AXIS_INDEX] * height;
rectanglePosition.lt = anchorLt;
// set right bottom point of rectangle
Coordinate anchorRb;
anchorRb.x = ptr[ANCHOR_RIGHT_BOTTOM_X_AXIS_INDEX] * width;
anchorRb.y = ptr[ANCHOR_RIGHT_BOTTOM_Y_AXIS_INDEX] * height;
rectanglePosition.rb = anchorRb;
// construct rectangle attribute
RectangleAttribute rectangleAttr;
rectangleAttr.position = rectanglePosition;
// set rectangle color
Color rectColor = { FACE_BOX_COLOR_R, FACE_BOX_COLOR_G,
FACE_BOX_COLOR_B };
rectangleAttr.color = rectColor;
// set rectangle thickness
rectangleAttr.thickness = FACE_BOX_BORDER_WIDTH;

// construct label attribute
TextAttribute textAttr;
std::stringstream labelContent;
int32_t scorePercent = score * SCORE_PRECENT;  // percent
labelContent << FACE_LABEL_TEXT_PREFIX << scorePercent
<< FACE_LABEL_TEXT_SUFFIX;
labelContent >> textAttr.text;
// set label position
Coordinate labelLb;
labelLb.x = anchorLt.x;
labelLb.y = anchorLt.y;
textAttr.position = labelLb;
// set label font type
textAttr.fontType = FACE_LABEL_FONT;
// set label color
Color labelColor = { FACE_LABEL_COLOR_R, FACE_LABEL_COLOR_G,
FACE_LABEL_COLOR_B };
textAttr.color = labelColor;
// set label font scale
textAttr.fontScale = FACE_LABEL_FONT_SIZE;
// set label thickness
textAttr.thickness = FACE_LABEL_FONT_WIDTH;

// push back
RectangleLabelAttribute rectLabel;
rectLabel.rectAttr = rectangleAttr;
rectLabel.textAttr = textAttr;
rectangleLabelAttr.emplace_back(rectLabel);
// construct input Image Attr
ImageAttribute srcImgAttr;
srcImgAttr.data = imgVec[ind].img.data;
srcImgAttr.height = height;
srcImgAttr.imageFormat = YUV420SPNV12;
srcImgAttr.size = imgSize;
srcImgAttr.width = width;
// construct output
std::shared_ptr<u_int8_t> dstImgBuffer = std::shared_ptr<uint8_t>(
new uint8_t[out.size], std::default_delete<uint8_t[]>());
// call DrawRectangelAndLabel interface
int32_t osdRet = DrawRectangelAndLabel(srcImgAttr,
rectangleLabelAttr,
dstImgBuffer);
