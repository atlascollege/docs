通过VLC播放实时视频

通过VLC播放实时视频

前提条件

  • Atlas DK开发板上安装gstreamer。

    参考如下命令：

    sudo apt-get install libgstreamer1.0-0 gstreamer1.0-plugins-base gstreamer1.0-plugins-good gstreamer1.0-plugins-bad gstreamer1.0-plugins-ugly gstreamer1.0-libav gstreamer1.0-doc gstreamer1.0-tools

    说明：
    此处需要Atlask DK连接外部网络，并在Atlask DK开发板上开启apt-get源，确保apt-get命令可用。

  • 主机安装vlc视频工具，且可以和Atlas DK开发板网络通讯正常。

    说明：
    此处主机可以使用UI Host或任一台可以和Atlas DK开发者板正常网络连接的主机。
    VLC为第三方播放软件，非华为自研软件。

观看视频

预置条件准备好后，在Atlas DK开发板上执行如下命令。

sudo ./ascendcamera -v -w 1280 -h 720 --fps 10 -o - | gst-launch-1.0 -e -v fdsrc ! "video/x-h264,width=1280,height=720,framerate=10/1" ! h264parse ! flvmux ! tcpserversink host=10.10.10.2 port=5000

配置参数说明：

  • gstreamer中的参数值必须按照ascendcamera中设置的参数值进行配置。

    示例：ascendcamera中的-w -h --fps配置的参数值必须和gst-launch-1.0（gstreamer的工具）的width height和framerate保持一致。

  • gstreamer中的host参数为Atlas DK开发板的ip地址，port为Atlas DK开发板上任一未被占用的端口号。

在主机的vlc视频工具中进行如下操作：“媒体->打开网络串流->在“请输入网络 URL:”中填写tcp://10.10.10.2:__5000/->播放，配置好参数后观看实时视频。
