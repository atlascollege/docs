facialrecognition

smartcity ->facialrecognition

人脸识别sample工程。

https://github.com/Ascend/ascenddk

Face_detection_model

Face_detection_model

人脸检测Sample模型，此模型为转化后的适配Ascend 310芯片的模型。

若用户使用Obtain the source code中的原始网络模型通过Mind Studio模型转化功能转化，转化时保持默认配置即可。

转化时会提示转化失败，这是只需要在最后一层的下拉框中选择SSDDetectionOutput然后点击Retry重新转换即可。

https://github.com/Ascend/models

VanillaCNN_Model

VanillaCNN_Model

人脸面部关键点标记模型，此模型为转化后的适配Ascend 310芯片的模型。

若用户使用Obtain the source code中的原始网络模型通过Mind Studio模型转化功能转化，转化时配置请将“Optional Options > Input Image Preprocess”设置为“Off”。

https://github.com/Ascend/models

Sphereface_Model

SpherefaceModel

人脸特征向量获取模型，此模型为转化后的适配Ascend 310芯片的模型。

若用户使用Obtain the source code中的原始网络模型通过Mind Studio模型转化功能转化，转化时配置请参考图1。

https://github.com/Ascend/models

Presenter Server

common->presenter->server

Presenter服务端，负责人脸识别的结果展示。

https://github.com/Ascend/ascenddk

图 1 SpherefaceModel模型转化配置参考

  • Input Shape中的N:8表示人脸识别程序，每次处理8张人脸。
  • Input Image Format：输入图片的格式，此处选择RGB888_U8。
  • Input Image Size[W|H]：因为输入图片格式为RGB8888_U8，此处不需要做128*16对齐，直接使用模型要求的宽和高即可，即96与112。
  • Mean Less[B|G|R]：此模型训练使用的图片的均值，可从此模型的sphereface_model.prototxt文件中获取，获取地址：Link。
  • Multiplying Factor[B|G|R]：此模型训练使用的图片的乘系数，可从此模型的sphereface_model.prototxt文件中获取，即scale的值，获取地址：Link。
