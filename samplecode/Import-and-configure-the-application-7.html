Import and configure the application

Import and configure the application

工程准备

  1. 参考Obtain the source code以Mind Studio安装用户下载facialrecognition 源码包到Mind Studio服务器任一目录，facialrecognition 文件夹包含如下子目录，其中子目录facialrecognitionapp为工程源码，MyDataset子目录为数据集，MyModel子目录为模型。

    facialrecognition 
├─facialrecognitionapp
├─MyDataset 
└─MyModel
└─run.sh

  2. 将face_detection_model，sphereface_model，vanillacnn_model模型包放入facialrecognition源码包的MyModel子目录中。

    facialrecognition
├─facialrecognitionapp
├─MyDataset 
└─MyModel 
   └─face_detection_model
   └─sphereface_model
   └─vanillacnn_model
└─run.sh

  3. 修改“facialrecognition > facialrecognitionapp > graph.config“文件。

    • 修改“engine_name“为“Mind_camera“的“data_source“的items的值。

      engines {
    id: 424
    engine_name: "Mind_camera"
    side: HOST
    thread_num: 1
    so_name: "./libMind_camera.so"
    ai_config {
     ...
      items {
        name: "data_source"
        value: "Channel-2"   //摄像头所属的Channel，有Channel-1和Channel-2两种取值
      }
      ...

      查询摄像头所属的Channel的方法如下：

      1. 在Mind Studio界面中单击“Tools > Atlas DK Configuration“，进入“Atlas DK Configuration“界面。如图1所示。

        图 1 Atlas DK配置界面

        说明：
        如果配置界面中无开发板连接，请参考《开发板使用指导》添加连接。

      2. 选中使用的开发板连接，单击“Connect“，查看开发板连接状态，如图2所示。

        图 2 Atlas DK状态界面

        图中“Camera2“的状态为“Online“，表示当前摄像头的“Data source“的值是“Channel-2“。

    • 修改“engine_name“为“face_register“的presenter_server信息及app_name。

      如下所示，分别修改“presenter_server_ip“、“presenter_server_port“与“app_name“对应的“value“。

        engines {
    id: 966
    engine_name: "face_register"
    side: DEVICE
    thread_num: 1
    so_name: "./libface_register.so"
    ai_config {
      items {
        name: "presenter_server_ip"
        value: "192.168.4.32"          //Presenter Agent服务端的IP地址
      }
      items {
        name: "presenter_server_port"
        value: "7008"                 //Presenter Agent服务端端口号
      }
      items {
        name: "app_name"
        value: "sample"              //Presenter Server界面上展示的此app的名称，用户自定义即可                    
      }
    }
  }

      说明：
      “presenter_server_ip“与“presenter_server_port“配置为3中config.conf文件中配置的值。
