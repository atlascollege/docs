Enable the Web Service on the UI Host

Enable the Web Service on the UI Host

  1. 将Presenter Server源码包（即server文件夹）上传到Mind Studio所在Ubuntu服务器任意目录，并将server文件夹的属组设置为MindStudio安装用户。

    在root用户下执行如下命令设置文件夹属组：

    chown -R uesrname:usergroup server。

    说明：

    • username为Mind Studio安装用户。
    • usergroup为Mind Studio安装用户所属的用户组。

    Presenter Server在ascenddk源码包中的目录参见表1。

  2. 安装Presenter Server依赖包。

    在目录“ascenddk/common/presenter/server”下执行如下命令进行依赖包的安装。

    pip3 install -r requirements

    说明：
    因为Presenter Server源码基于python3.5，如果环境中不存在python3.5，请自行安装。
    可执行 python3 --version命令查看系统中的python3版本。

  3. 修改“config.conf”配置文件，“config.conf”所在位置如下。

    presenter 
├─server 
  ├─video_analysis 
    ├─config 
      └─config.conf

    “config.conf”配置文件内容显示如下所示：

    [baseconf] 
# A socket server address to communicate with presenter agent 
# Please ensure that the port does not conflict, only support Ipv4 
presenter_server_ip=xx.xx.xx.xx
presenter_server_port=7004 

# A http server address, you can visit the website by "http//web_server_ip:web_server_port". 
# Only support Chrome now. 
web_server_ip=xx.xx.xx.xx 
web_server_port=7005 

# Storage path of video_analysis information
# Please ensure the storage_dir exist before running the program
storage_dir=/home/username/videodate/presenter/video_analysis

# Maximum number of supported apps, range is 1-10
max_app_num=2

# Reserved space. If remaining space of storage_dir < reserved_space,
# stop storing data and force stopping app
# Unit MByte, minimum value not less than 1024
reserved_space=1024

    • “presenter_server_ip“和“preset_server_port“分别是Presenter Agent的服务端IP和端口。

      说明：
      IP地址需要配置成Mind Studio连接开发板的实际IP地址，需要和Atlas DK开发板的IP在同一网段，默认端口号为7004。
      此端口需要为未被占用的端口号，不同的应用建议使用不同的端口号。

    • “web_server_ip“和“web_server_port“分别是web服务端的IP和端口。

      说明：
      配置为Mind Studio所在Ubuntu服务器的IP地址，此IP地址为用浏览器访问Presenter Server服务的IP地址，默认端口号为7005。
      此端口需要为未被占用的端口号，不同的应用建议使用不同的端口号。

    • “storage_dir“：视频解析数据的存储路径，此路径用户可以自定义，需要为Mind Studio所在服务器中已存在的绝对路径，此路径Mind Studio安装用户需要有读写权限。

    • “max_app_num“：支持的最大app的数量，即图1中Video_app的数量，取值范围为[1,10]。
    • “reserved_space“：“storage_dir“路径中允许存储的数据大小的最大值，单位为MB，此值最小不能低于1024。

  4. 启动Presenter Server。

    进入到“server”目录下，执行如下命令在后台启动Presenter Server主程序。

    nohup python3.5 presenter_server.py --app video_analysis &

    执行脚本所在目录如下：

    presenter 
├─server 
  └─presenter_server.py

    如图3-2所示，表示presenter_server的服务启动成功。

    图 1 presenter server进程启动成功

  5. 登录Presenter Server主页，URL为http://_web\_server\_ip_:_web\_server\_port_，进入主页。

    • web_server_ip为3config.conf中配置的web_server_ip。
    • web_server_port为3中config.conf中配置的web_server_port，默认端口为7005。
