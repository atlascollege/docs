Compiling and Running

Compiling and Running

  1. 双击左侧Project菜单树，打开“facedetectionapp.mind“文件，在画布区域的下方单击Run按钮。

  2. 填写Atlas DK开发板所在IP地址（图1IP仅为示例，请按照实际填写），单击run按钮。

后续处理

人脸检测应用运行后，在Atlas DK开发板上会存在相应的进程，可以在开发板上执行如下命令查找运行的应用程序对应的进程ID。

ps -ef|grep hiai

如下所示：

$ ps -ef|grep hiai
root      9478  2026  0 23:00 ?        00:00:00 sh -c /root//HIAI_PROJECTS/workspace_hiai_studio//facedetectionapp/out/hiai_workspace_hiai_st                      udio_facedetectionapp_main
root      9479  9478  0 23:00 ?        00:00:00 /root//HIAI_PROJECTS/workspace_hiai_studio//facedetectionapp/out/hiai_workspace_hiai_studio_f                      acedetectionapp_main
root      9577  8024  0 23:02 pts/0    00:00:00 grep --color=auto hiai

如需停止人脸检测应用程序，则执行如下命令将相应的进程停止即可。

sudo kill -9 9478

sudo kill -9 9479

说明：
此处9478与9479为人脸检测进程ID示例。
