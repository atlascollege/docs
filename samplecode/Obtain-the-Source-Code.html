object_detection_model

object_detection_model

目标检测模型，此模型为转化后的适配Ascend 310芯片的模型。

若用户使用How it works中的原始网络模型通过Mind Studio模型转化功能转化，转化时保持默认配置即可。

转化时会提示转化失败，这是只需要在最后一层的下拉框中选择SSDDetectionOutput然后点击Retry重新转换即可。

https://github.com/Ascend/models
