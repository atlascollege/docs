Pre-installation

Pre-installation

运行Atlas DK开源Application前，需要按照此章节进行Presenter Agent、EZDVPP、OSD的相关环境配置。

  1. 获取源码包。

    从https://github.com/Ascend/ascenddk中将common文件夹以Mind Studio安装用户下载至Mind Studio所在Ubuntu服务器的任意目录。

    如果common文件夹属组不为Mind Studio安装用户，设置方法如下：

    切换到root用户下执行如下命令：

    chown -R uesrname:usergroup common

    说明：

    • username为Mind Studio安装用户。
    • usergroup为Mind Studio安装用户所属的用户组。

  2. 以Mind Studio安装用户登录Mind Studio所在Ubuntu服务器，并设置环境变量DDK_HOME。

    vim ~/.bashrc

    执行如下命令在最后一行添加DDK_HOME的环境变量。

    export DDK_HOME=/home/XXX/tools/che/ddk/ddk

    说明：

    • XXX为Mind Studio安装用户，/home/XXX/tools为DDK默认安装路径。
    • 如果此环境变量已经添加，则此步骤可跳过。

    输入:wq!保存退出。

    执行如下命令使环境变量生效。

    source ~/.bashrc

  3. 执行如下命令安装pexpect。

    pip3 install pexpect

  4. 进入1下载的common目录，并执行如下命令进行环境准备操作。

    python3 common_install.py

    如图1所示，根据脚本回显，分别输入：

    • Atlas DK开发板的IP地址。
    • Atlas DK开发板的ssh登录用户名，此处请填写普通用户（非root用户），默认为hisilicon用户
    • Atlas DK开发板的普通用户密码，其中hisilicon用户的默认密码为Mind@123
    • Atas DK开发板的ssh端口号，默认为22
